TrpB COMM Domain Dynamics via Well-Tempered MetaDynamics

Enhanced sampling of Open → Closed conformational transitions in tryptophan synthase β-subunit using path collective variables

Duration Summer 2026
PI Prof. Anima Anandkumar
Institution California Institute of Technology
Award $8,110 SURF Fellowship

Research Question

What can dynamics tell us that sequence generation and static structure prediction cannot? Engineered TrpB variants like GenSLM-230 and NdTrpB share nearly identical predicted structures, yet they exhibit dramatically different catalytic behavior. This project uses well-tempered MetaDynamics to map the free energy landscape of the COMM domain (residues 97–184), which gates the Open → Partially Closed → Closed conformational transitions essential for substrate channeling and catalysis.

Why Dynamics Matters

The long-term goal is to use enhanced sampling as a physics layer inside a generative TrpB design pipeline. Three concrete integration targets motivate this work:

  1. Functional divergence: Explain why GenSLM-230 and NdTrpB behave differently despite near-identical AlphaFold structures
  2. Physics-based filter: Test whether MetaDynamics-derived free energy barriers can serve as a conformational filter for D-Trp selectivity screening
  3. Surrogate data export: Extract trajectory-derived features for training a faster ML surrogate that can score large candidate libraries

Methodology

Well-tempered MetaDynamics with PLUMED2 path collective variables, benchmarked against the Maria-Solano et al. JACS 2019 study of Pyrococcus furiosus TrpS:

  • Path CVs: s(R) = progress along the O→C path; z(R) = deviation from the path. Reference path constructed as a 15-frame Cα interpolation from 1WDW (open) to 3CEP (closed)
  • MetaDynamics parameters: Gaussian height 0.15 kcal/mol, bias factor γ=10, deposition every 500 steps
  • Conventional MD baseline: 500 ns unbiased production run (AMBER 24p3, pmemd.cuda) to establish equilibrium dynamics before applying bias
  • Force field: ff14SB protein + GAFF/RESP for PLP cofactor (pyridoxal phosphate parameterization via antechamber)
  • Solvent & temperature: TIP3P explicit water, 350 K (thermophilic enzyme)

Systems Studied

System Description
PfTrpS(Ain) Primary benchmark: P. furiosus tryptophan synthase, internal aldimine state
5DVZ / 5DW0 / 5DW3 Crystallographic TrpB structures capturing different catalytic intermediates
GenSLM-230 AI-designed variant, comparison target for functional divergence analysis
NdTrpB Natural variant, structurally similar to GenSLM-230 but functionally distinct

Reaction intermediates along the catalytic cycle: Ain (internal aldimine), Aex1 (external aldimine), A-A (amino-acrylate), Q2 (quinonoid).

Campaign Architecture

The project follows a two-layer design separating calibration from scientific exploration:

  1. Benchmark reproduction: Validate the MetaDynamics setup against published PfTrpS literature (JACS 2019)
  2. Mechanistic comparison: Compare COMM domain dynamics between sequence pairs (GenSLM-230 vs NdTrpB)
  3. Physics-based filter design: Define dynamics-derived criteria for D-Trp selectivity
  4. Surrogate data bootstrap: Export traceable features from trajectories for downstream ML models

Computational Pipeline

A 6-stage automated pipeline manages the full workflow on UNC Longleaf HPC:

  • Profiler: Extract structured campaign facts from papers and planning documents into manifests
  • Librarian: Gather and audit external resources (PDBs, supplementary information, references)
  • Janitor: Normalize workspace, prepare campaign directories, stage simulation inputs
  • Runner: Generate and execute deterministic run steps (Slurm jobs, dry-run sequences)
  • Skeptic: Validate artifacts; separate operational success from scientific validity
  • Journalist: Write concise campaign reports summarizing outcomes and blockers
